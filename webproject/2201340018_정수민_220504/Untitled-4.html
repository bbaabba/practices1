컨텐츠 표시를 위한 section
header영역 : 홈페이지의 각 메뉴를 표시
각 대목차들은 onmouseenter이벤트로
마우스가 올라가면 소목차를 펼친다
대목차를 클릭하면 대목차 내의 첫번째 소목차로 페이지를 section구역에 띄운다.
width=100%, height = 15%예상
펼쳐진 소목차를 클릭하면 section은 클릭한 소목차로 이동한다.
nav 영역 : 현재 페이지의 대목차 내의 소목차 메뉴들을 띄운다
소목차마다 링크를 걸어 클릭하면
section 구역에 해당 소목차의 컨텐츠를 띄운다
left방향에 위치할것이며 width = 20%, height = 80%예상
section 영역 : 소목차의 html화면을 띄우는 영역.
나머지 width 60%, height 65%의 영역예상
